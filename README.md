Safety Monitoring System (Computer Vision)

A real-time Computer Vision-based Safety Monitoring System built using Python, OpenCV, and PyTorch to detect safety violations and potential hazards in industrial/workplace environments.

🚀 Features

🎥 Real-time video processing using OpenCV

🧠 Deep learning-based hazard detection (CNN models)

⚠️ Detects unsafe conditions from live video frames

📊 Frame-by-frame analysis pipeline

🔔 Scalable for alerts & monitoring systems

🛠️ Tech Stack

Python

OpenCV

PyTorch

NumPy

Matplotlib (for visualization)

📂 Project Structure

safety_monitor/

│── venv/                # Virtual environment (ignored)

│── models/              # Trained models (if any)

│── data/                # Dataset (if used)

│── main.py              # Main execution file

│── requirements.txt     # Dependencies

│── README.md            # Project documentation

⚙️ Installation

Clone the repository:

git clone https://github.com/Ayesha037/safety_monitor.git

cd safety_monitor

Create virtual environment:

python -m venv venv

venv\Scripts\activate   # Windows

Install dependencies:

pip install -r requirements.txt

▶️ Usage

Run the main script:

python main.py

The system will start processing video input and detect safety-related anomalies.
